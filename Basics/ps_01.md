Pseudocode for Area of Square:-

PROGRAM  Square_Area

    READ length
    PRINT "Area of square is"  length * length

Definations:-
PROGRAM and END PROGRAM are reserved  words in pseudocode used at the start and  end of the program respectively.
READ and PRINT are used to get input from user and display output respectively.
